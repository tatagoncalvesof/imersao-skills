# Skills da Imersao IA + Ads — By Tata Goncalves

Bem-vindo! Essas sao as skills exclusivas que voce recebeu por participar da Imersao IA + Ads.

## O que sao Skills?

Skills sao "super poderes" que voce instala no Claude Code. Quando voce digita o nome da skill, o Claude Code automaticamente sabe EXATAMENTE o que fazer, com templates profissionais e metodologia testada.

## Skills incluidas

| # | Skill | O que faz |
|---|-------|----------|
| 1 | `/skill-claude-md-builder` | **COMECE POR AQUI!** Cria seu CLAUDE.md interativamente — a IA te faz perguntas estrategicas e monta o arquivo automaticamente |
| 2 | `/skill-copy-builder` | **CRIE SUA PROPRIA SKILL DE COPY!** Entrevista voce sobre negocio, publico, tom, e referencias (Russell, Hormozi, Bencivenga, Sexy Canvas) e gera uma skill de copy personalizada |
| 3 | `/skill-persona-profunda` | Gera persona completa com 30 dimensoes psicologicas + Buyer Persona + ICP + Mapa de Empatia |

## Ordem recomendada

1. **Primeiro:** `/skill-claude-md-builder` — Configura a IA pro seu negocio
2. **Segundo:** `/skill-copy-builder` — Cria sua skill de copy personalizada
3. **Terceiro:** `/skill-persona-profunda` — Gera persona profunda do seu publico

## Como instalar (3 passos)

### 1. Baixe este repositorio

Clique no botao verde **"Code"** > **"Download ZIP"** e descompacte no seu computador.

Ou se tiver git:
```bash
git clone https://github.com/tatagoncalvesof/imersao-skills.git
```

### 2. Copie as pastas de skills para o Claude Code

```bash
# Mac/Linux:
cp -r skill-claude-md-builder ~/.claude/skills/
cp -r skill-copy-builder ~/.claude/skills/
cp -r skill-persona-profunda ~/.claude/skills/

# Windows (PowerShell):
Copy-Item -Recurse skill-claude-md-builder $env:USERPROFILE\.claude\skills\
Copy-Item -Recurse skill-copy-builder $env:USERPROFILE\.claude\skills\
Copy-Item -Recurse skill-persona-profunda $env:USERPROFILE\.claude\skills\
```

### 3. Reinicie o Claude Code

Feche e abra o Claude Code novamente. As skills ja estarao disponiveis!

## Como usar

Dentro do Claude Code, basta digitar o nome da skill:

```
/skill-claude-md-builder
```

**Comece por essa!** Ela vai te guiar pra criar seu CLAUDE.md (o manual de instrucoes da IA pro seu negocio).

Depois:
```
/skill-copy-builder
```
Cria sua skill de copy personalizada — a IA te entrevista e gera tudo automaticamente.
